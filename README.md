# Analise_de_Precos_de_Combustivel_no_Brasil
Projeto de análise de dados com base pública da ANP, realizando ETL completo no Power BI, incluindo limpeza, transformação e modelagem dos dados. O resultado foi um dashboard interativo com segmentações e bookmarks para suporte à tomada de decisão. As ferramentas utilizadas foram Power BI, Power Query, DAX e Excel para validação.
